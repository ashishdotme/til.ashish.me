# Dotnet Common commands

Select words

```
ctrl + w
```

Go to matching brace

```
ctrl + ]
```

Enter line above

```
ctrl + enter
```

Enter line below

```
ctrl + shift + enter
```

Go to top of the page

```
ctrl + home
```

Go to the bottom of the page

```
ctrl + end
```

Quickle Navigate files

```
ctrl + e
```

Search Everywhere

```
ctrl + n
```

Navigate to the usage of object

```
alt + f7
```
